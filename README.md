# modoo-shop
- MVC 패턴을 활용한 JSP 팀 프로젝트 with 웹 크롤링
- 다양한 보드게임들을 쉽게 찾아 보고 구매할 수 있는 사이트

# 구현 기술 스택
- JSP
- Oracle (11g XE)
- Java, HTML, CSS, JS(Ajax)
- Apache Tomcat

- JDBC
- Eclipse
- Dynamic Web Project
